Helpful bookmarklets for PAs

To install, drag link onto your brower's bookmark bar. Click to run.

You may be asked to give the site permission to access the clipboard the first time you run one on a given page.

🗝 Open Key - checks clipboard for a CMM key, if it finds one it opens the request in a new tab

🗝 Copy Key - on a CMM request page, copies the key to the clipboard

Fax🛠 - checks the clipboard for a phone/fax number, if it finds one it strips all punctuation and spaces (useful for sending efaxes to pharmacies)
